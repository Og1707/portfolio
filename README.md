## Estructura

```
osman-portfolio/
├── frontend/     # React + Vite
└── backend/      # FastAPI + Python
```

## Inicio Rápido

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
uvicorn app.main:app --reload
```

## Variables de Entorno

Ver `backend/.env.example` para configuración requerida.
